# Genius Ad District Accessibility Documentation

## Our Commitment to Accessibility

At Genius Ad District, we are committed to ensuring our website is accessible to all users, regardless of abilities or disabilities. We strive to comply with Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

## Key Accessibility Features

### Keyboard Navigation

- All interactive elements are accessible via keyboard
- Visible focus indicators for keyboard users
- Logical tab order that follows the visual layout
- Skip navigation link at the top of each page
- No keyboard traps in modal dialogs or complex components

### Screen Reader Support

- Semantic HTML structure with proper landmarks
- ARIA roles, states, and properties where appropriate
- Descriptive link text and button labels
- Form inputs with associated labels
- Proper heading hierarchy for content structure
- Alt text for all meaningful images

### Color Contrast and Visual Design

- All text meets WCAG AA color contrast requirements (4.5:1 for normal text, 3:1 for large text)
- Color is not used as the only means of conveying information
- User interface elements have sufficient contrast against their backgrounds
- Text can be resized up to 200% without loss of content or functionality

### Media Accessibility

- Videos include captions
- Audio content has transcripts available
- Video players include accessible controls
- Non-text content has appropriate text alternatives

### Responsive Design

- Content is accessible on various screen sizes and orientations
- Touch targets are sufficiently large for mobile users
- Content reflows appropriately when zoomed

## Accessibility Components

Our site includes several specialized components to enhance accessibility:

- **FocusTrap**: Prevents keyboard focus from leaving modal dialogs
- **SkipNavigation**: Allows keyboard users to bypass navigation menus
- **AccessibleMedia**: Ensures images have proper alternative text
- **VideoPlayer**: Provides captions and transcript support for videos
- **AudioPlayer**: Includes transcript support and accessible controls

## Known Limitations

While we strive for full compliance with WCAG 2.1 AA standards, we are continuously working to improve our site. If you encounter any accessibility issues, please contact us.

## Feedback

We welcome feedback on the accessibility of our website. If you encounter barriers or have suggestions for improvement, please email us at accessibility@geniusaddistrict.com.

## Testing

Our accessibility features are tested using a combination of:

- Automated testing tools (Axe, WAVE)
- Manual keyboard testing
- Screen reader testing with NVDA and VoiceOver
- User testing with individuals with various disabilities
